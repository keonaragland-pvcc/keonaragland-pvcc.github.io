Operating Systems Terminology

TERM DEFINITION
LAN consists of a collection of computers in a single building or building complex
WAN links machines over a greater distance—perhaps in neighboring cities or on opposite sides of the world.
Bus network machines are all connected to a common communication line
Star network One machine serves as a central focal point to which all the others are connected
CSMA/CD Ethernet dictates that each message be broadcast to all the machines on the bus
CSMA/CA Wi-Fi wireless networks adopt the policy of trying to avoid collisions rather than trying to detect them
Repeater device that passes signals back and forth between the two original buses (usually with some form of amplification) without considering the meaning of the signals
Bridge destination address that accompanies each message and forwards a message across the connection only when that message is destined for a computer on the other side
Switch bridge with multiple connections, allowing it to connect several buses rather than just two
Router special purpose computers used for forwarding messages
Gateway The “point” at which one network is linked to an internet
P2P peer-to-peer: processes that provide service to and receive service from each other
Distributed system consist of software units that execute as processes on different computers
Cloud computing whereby huge pools of shared computers on the network can be allocated for use by clients as needed, is the latest trend in distributed systems
Tier 1 ISPs that consist of very high-speed, high-capacity, international WANs
Tier 2 ISPs that tend to be more regional in scope and less potent in their capabilities
Tier 3 : Access to this core is usually provided by an intermediary
ICANN nonprofit corporation established to coordinate the Internet’s operation https://www.icann.org/
DNS name servers are used as an Internet-wide directory system
TLD used to reflect the domain’s classification, which in this case is “educational,” as indicated by the suffix
Country-code TLDS creation of new two-letter TLDs for each specific country https://www.worldstandards.eu/other/tlds
FTP application for accessing another computer from a great distance
HTTP application layer protocol in the internet protocol suite model for distributed, collaborative, hypermedia information systems
SMTP defines a way that two computers on the network may interact when transmitting an email message from one host to the other
URL contains the information needed by a browser to contact the proper server and request the desired document
Client-side activities activities if they are performed by a client
Server-side activities activities if they are performed by a server (such as a webserver)
Application Layer software units such as clients and servers that use Internet communication to carry out their tasks
Transport Layer accept messages from the application layer and to ensure that the messages are properly formatted for transmission over the Internet
Network Layer in charge of maintaining the router’s forwarding table and using that table to determine the direction in which to forward packets
Link Layer responsibility of transferring the packet
Packet-switching network digital data is sent by breaking the data into blocks of bits
Fault tolerant a process that enables an operating system to respond to a failure in hardware or software
Redundant between two points on the Internet increases the reliability of the Internet and helps it scale to more devices and more people
Malware malicious software
Virus software that infects a computer by inserting itself into programs that already reside in the machine
Worm autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers
Trojan Horse program that enters a computer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim
Spyware software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack
Phising technique of obtaining information explicitly by simply asking for it
Ddosattack When the work of generating the attacks is distributed over many hosts
Firewall A primary prevention technique is to filter traffic passing through a point in the network
Spam Filter firewalls designed to block unwanted email
Proxy server software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server
Antivirus software used to detect and remove the presence of known viruses and other infections
Cryptography study of sending and receiving messages securely in the presence of adversaries
Public key encryption involves techniques by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure
