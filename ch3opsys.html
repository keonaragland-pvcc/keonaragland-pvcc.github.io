Operating Systems Terminology

TERM DEFINITION
Operating System Software that controls the overall operation of a computer.
Job Queue Storage organization in which objects (jobs) are ordered in a first-in, first-out fashion.
FIFO First In First Out.
Interactive Processing A terminal consisted of little more than an electronic typewriter by which the user could type input and read the computer’s response that was printed on paper.
Time-sharing Computer is forced to execute tasks under a deadline.
Multiprogramming Time is divided into intervals, and then the execution of each job is restricted to only one interval at a time.
Application Software Programs for performing tasks particular to the machine’s utilization.
System Software Provides the infrastructure that the application software requires, similar to a nation’s infrastructure providing the foundation for citizens' lifestyles.
Utility Software Consists of programs for performing activities fundamental to computer installations but not included in the operating system.
Graphical User Interface (GUI) Allows users to issue commands using common input devices.
Shell Communicates with users through textual messages using a keyboard and monitor screen.
Kernel Software components that perform the very basic functions required by the computer installation.
Device Driver Software units that communicate with controllers to carry out operations on peripheral devices.
File Manager Coordinates the use of the machine’s mass storage facilities.
Memory Manager Charged with the task of coordinating the machine’s use of main memory.
Virtual Memory large “fictional” memory space created by paging
Boot Strapping Procedure that transfers the operating system from mass storage into main memory when the machine is first turned on.
ROM A small portion of a computer’s main memory where the CPU expects to find its initial program, constructed from special nonvolatile memory cells.
Deadlock Condition in which two or more processes are blocked from progressing because each is waiting for a resource allocated to another.
Spooling Holding data for output at a later but more convenient time.
Super User Person gains highly privileged access to the operating system by identifying as the administrator during the login procedure.
Multi-Factor Authentication A security process in which the user provides two or more different factors to verify their identity.
